システムの仕組み

幾何生成 → 制約チェック/修復 → 人間による比較選択 → 進化、のループで設計空間を探索します。

1. 幾何生成

サイト座標と重み (x, y, z, w) から 3D Power Diagram を構築。各セルは凸多面体の n-gon フェースとして出力。

2. 制約と修復

6種類の制約をチェックし、違反した個体を自動修復。実現可能な形態のみを次の評価に進める。

3. 対話的選択

1世代6候補を Swiss-system トーナメント（9マッチ）で比較し、設計者が好みの形態を選ぶ。

4. 進化と系譜

勝者と高品質 Archive を親に次世代を生成。provenance により形態の由来を完全に追跡。
